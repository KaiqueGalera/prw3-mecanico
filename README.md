# Oficina Mecânica API 🔧

Uma API REST completa e profissional para o gerenciamento de consertos em uma oficina mecânica, desenvolvida como avaliação prática para a disciplina de **Programação para Web 3** no **IFSP São Carlos**.

A aplicação permite o controle total de ordens de serviço, incluindo dados do veículo, mecânico responsável e datas de entrada/saída, utilizando as melhores práticas de desenvolvimento Spring Boot.

## 🛠 Tecnologias
* **Java 21**
* **Spring Boot 3**
* **Spring Data JPA** (Camada de Persistência)
* **Flyway** (Gerenciamento de Migrations do Banco de Dados)
* **H2 Database** (Banco de dados em memória/arquivo para desenvolvimento)
* **Lombok** (Produtividade e código limpo)
* **Bean Validation** (Validação de dados de entrada)
* **Maven** (Gerenciamento de dependências)

## 🚀 Como Executar
1. Clone este repositório em sua máquina local.
2. Abra o projeto no **IntelliJ IDEA** (ou sua IDE de preferência).
3. Certifique-se de que todas as dependências do **Maven** foram sincronizadas.
4. Execute a classe `MecanicaApplication.java`.
5. A API estará disponível em `http://localhost:8080`.
